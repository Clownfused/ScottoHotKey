Order Helper by Scott Rippey - User's Guide

 

Quick Start Guide

Starting the program

Order Helper is a portable, stand-alone exe file. Double-click the file to get started.  The program will open in the background, and you will see a small green "H" icon in your task tray (next to the clock).  The program will continue running in the background; to exit the program, right-click the green icon and choose Exit.

To bring up the Order Helper window, hold down the Windows Key and press O.  I will refer to this shortcut as Win+O from now on.

Processing a Web Order

To process a web order, you must first open the web order in Firefox.  Internet Explorer is not currently supported.  Open the web order (https://www.griffinlab.com/catalog/admin/orders.php) and change the Order Status to Processing. Now Print the order.

Bring up the Order Helper window (shortcut: Win+O), and press the Web Order => button to import all the data from the web order.

Order Helper will now prompt you for the "Middle Digits" of the credit card.  You must login to Yahoo Mail to obtain these digits (http://mail.yahoo.com ).  Type the 8 middle digits into the prompt, or just copy them to the clipboard.

Now, the order window will appear, with as much information filled out as possible.  Double-check all the order information.  Be sure to click Apply if you make any changes!

Charging the Credit Card with VirtualMerchant

The next step is to charge the credit card.  If the Billing Address is not in the US, Order Helper will have a few different steps.  Please read the International Customer Instructions instead.

Open Virtual Merchant ( http://www.virtualmerchant.com ) and login.  On the left, click Credit Card then click Sale.

The box labeled Card Account Data should have a blinking cursor (if not, click it).  In the Order Helper window, click <= Virtual Merchant to automatically type out the credit card number.  Click Submit.

Now Virtual Merchant should display a list of fields to fill out.  Make sure the first box, Expiration Date, is selected.  In the Order Helper window, click <= Virtual Merchant again.

You will now see a prompt that asks "Do I need to paste the Customer Code and Sales Tax?"  Take a look at the Virtual Merchant window.  Right above the Description field, there will be both "Customer Code:" and "Sales Tax:", or just "Sales Tax:", or neither, depending on the credit card account.

Example:
Account Data:
Expiration Date(MMYY):
Amount:
CVV2 Indicator:
CVV2:
Customer Code:
Sales Tax:
Description:
Billing Address
............

Obviously, if you see both fields, choose Both, or if you just see Sales Tax, choose Sales Tax, or else choose None.

Once you make your selection, the fields will be automatically filled.  Double-check that the Amount is correct. Scroll to the bottom and click Process.

Assuming the card is approved, a new window will pop-up with the Credit Card Receipt.  Hit OK to print it.

On the Credit Card Receipt printout, write down the type of card (Visa, MC, Amex, Disc), the Web Order #, and sign it with your initials.

 

Putting Customer Information into PeachTree

Putting information into Peachtree is a two-step process.  First, you have to create a new customer.  Second, you have to create a sales order.

Open PeachTree, login, and click Customers -> New Customer.

In the Order Helper, click <= PeachTree to automatically enter the customer's name into the CustomerID field.

Check to see if the customer already exists in the database. If the customer does already exist, double-check that the address matches the Order Helper address. Then, close this window and continue to the next step, Creating a Sales Order in PeachTree.

If the customer does not exist, then click <= PeachTree again.  This will auto-type the customer's information.  Click Save and close the window.

Creating a Sales Order in PeachTree

In PeachTree, click Sales Orders -> New Sales Order. 

In the Order Helper, click <= PeachTree again.  This will fill in the CustomerID.  Make sure the correct CustomerID is selected.

In the Order Helper, click <= PeachTree one last time.  This will start filling out the entire sales order.  This process might take a short while; just wait for it to finish.

Double-check ALL information, including Shipping Method and the Order Total, to see if they match the original order.  If they do, click the Print button, make sure Sales Order Picking List is the selected form, and click Print.

Finishing Up

To finish the Web Order, you should staple all 3 printouts:  the Sales Order Picking List, the Web Order, and the Credit Card Receipt (in that order).  The Sales Order now goes into the production room.  You are now finished with the Web Order.

 

 

 

 

Other Tasks

Importing and Exporting Customer Information into ACT!

To export or import customer information between ACT! and Order Helper, first make sure ACT! is open and has the customer info displayed.

To export the Order Helper information into ACT!, you must first check if the contact exists.  Click Lookup -> Last Name...  and search for the customer's last name.  The shortcut for this is Win+F.  If the customer already exists, you should double-check the information, and then you are done.
If the customer does not already exist, create a new ACT! contact by clicking Contacts -> New Contact.  Then, in the Order Helper, click <= ACT! to fill in all the fields.  Click Save.

To import the customer information from ACT! into Order Helper, bring up a customer in ACT!.  Open a new Order Helper window (Win+O), and click ACT! => to import all data.

 

 

 

Shortcuts

A good number of shortcuts exist for making order processing faster and easier.

To fix capitalization errors:
If a Company, Name, Address, City, or Country has capitalization errors, click the corresponding label to fix the errors.  For example, if the Name was joHN smiTH, then click the Name label and the text will change to John Smith.  Click it again to undo.

If the Description field is blank, click the Description label to insert a description based off the customer's last name and the date.  This is useful for phone orders.

Keyboard Shortcuts

Shortcut: Name: Applies to the following windows: Action:
Win+C Copy Info ACT Customer Info, PeachTree Customer Info, Web Orders Imports all data into a new Order Helper window.  Same thing as hitting Web Order =>, ACT! =>, or PeachTree =>.
Win+V Paste Info ACT Customer Info, PeachTree Customer Info, PeachTree Sales Order, Virtual Merchant Exports all data from the Order Helper into the application.  Same thing as <= Virtual Merchant, <= ACT!, or <= PeachTree.
Win+F Find Customer ACT Customer Info Find / lookup all users with the customer's last name
